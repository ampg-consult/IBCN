# IBCN Branding & Asset Guide

## 1. Logo & Iconography
- **Primary Logo**: A stylized "B" integrated with circuit-like network elements.
- **App Icon**: 512x512 PNG. The stylized "B" on a `Charcoal (#121212)` or `Deep Blue (#0D47A1)` background.
- **Style**: Futuristic, minimalist, and tech-inspired.

## 2. Color Palette
| Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Primary** | `#0D47A1` | Headers, Primary Buttons, Main Branding |
| **Accent** | `#00E5FF` | Progress bars, active states, highlights |
| **Secondary** | `#1976D2` | Secondary actions, sub-headers |
| **Neutral Dark** | `#121212` | Backgrounds (Dark Mode), Text (Light Mode) |
| **Neutral Light** | `#F5F5F5` | Backgrounds (Light Mode), Card surfaces |

## 3. Typography
- **Headers**: `Montserrat Bold`
- **Body & Buttons**: `Roboto Regular/Medium`

## 4. UI Patterns
- **Cards**: Elevation 4dp, rounded corners (12dp).
- **Buttons**: Medium rounded corners, uppercase labels for primary actions.
- **AI Feedback**: Use subtle `Electric Cyan` glows or pulse animations to indicate AI activity.
